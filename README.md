로봇 자동 물류 관리를 위한 통합 시스템의 안드로이드 애플리케이션 프로젝트입니다.

## 📱 프로젝트 소개
이 프로젝트는 물류 현장에서 사용되는 AGV(무인 운반 로봇)와 SCARA 로봇의 상태를 실시간으로 모니터링하고 제어하기 위해 개발된 안드로이드 앱입니다.

## ✨ 주요 기능
- **서버 통신**: 실시간 데이터 송수신을 통한 시스템 제어
- **물품 확인**: 현재 운송 중인 물품의 상태 확인
- **AGV 맵 내 위치 추적**: AGV의 실시간 위치 시각화 및 경로 이동 파악
- **실시간 스트리밍**: 현장 카메라를 통한 실시간 모니터링

## 🛠 기술 스택
- **언어**: Kotlin/Java (안드로이드)
- **통신**: Socket/Firebase (서버 통신)
- **라이브러리**: OpenCV (영상 처리)
